## 📖 About the Project

This project focuses on **preprocessing and analyzing the Netflix Prize Dataset** using Python. The dataset contains millions of movie ratings submitted by Netflix users.

The primary goal is to transform the raw data into a clean and structured format, perform exploratory data analysis (EDA), and prepare it for developing an efficient movie recommendation system.

---

## 🎯 Objectives

* Import and clean the Netflix dataset
* Handle missing and invalid values
* Extract Movie IDs from raw data
* Assign ratings to corresponding movies
* Perform Exploratory Data Analysis (EDA)
* Generate meaningful insights
* Prepare data for recommendation algorithms
